English to Urdu Translator - Offline (Argos Translate)

1. Install required libraries:
   pip install -r requirements.txt

2. Download the translation model:
   https://www.argosopentech.com/argospm/translate-en_ur.argosmodel

3. Place the downloaded model in this folder.

4. Run the install script:
   python install_model.py

5. Start the Streamlit app:
   streamlit run app.py

✔ Supports short & long English text
✔ Translates proper names correctly
✔ Works 100% offline
